# ROI Model

Primary value hypothesis: reduce wet-lab spend by eliminating low-evidence candidates before synthesis/procurement and by exporting assay packets that shorten CRO handoff time.

Measure:
- Number of candidates reviewed.
- Number promoted to wet-lab after two-person gate.
- Assay packet preparation time saved.
- Failed-assay avoidance from ADMET/selectivity/synthesis flags.
- Recalibration improvement after imported assay results.
