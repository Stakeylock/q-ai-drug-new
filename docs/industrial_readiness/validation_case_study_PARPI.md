# Validation Case Study: PARP1/PARP2

Goal: validate PARP inhibitor workflows across PARP1 target engagement, PARP-family selectivity, cellular PARylation response, and DNA-damage context.

Reference comparators: olaparib, rucaparib, niraparib.

Critical risks: PARP-family selectivity, genotoxicity interpretation, trapping mechanism, permeability, and hematologic safety hypotheses.
