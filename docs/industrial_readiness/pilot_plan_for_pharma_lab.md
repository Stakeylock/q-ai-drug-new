# Pilot Plan For Pharma Lab

Pilot duration: 6-10 weeks.

Scope:
1. Select one oncology target family and 50-200 internal or public molecules.
2. Run computational triage and generate assay packets.
3. Approve a small wet-lab shortlist through two-person gates.
4. Import assay results.
5. Compare predicted rank, uncertainty, and applicability-domain labels against experimental outcomes.
6. Review audit logs, exports, and governance gaps.
