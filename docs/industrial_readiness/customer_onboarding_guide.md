# Customer Onboarding Guide

1. Define research-use scope and prohibited claims.
2. Configure organization, users, roles, and artifact storage.
3. Load target set, receptor assets, known inhibitors, and assay history.
4. Upload or select ligand libraries.
5. Run computational pipeline and inspect candidate dossiers.
6. Generate wet-lab assay packets.
7. Apply decision gates and electronic signatures.
8. Import assay results and review active-learning summary.
9. Export audit logs and readiness evidence for sponsor review.
